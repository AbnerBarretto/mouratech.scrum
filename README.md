# Belo Eventos (Site Estático)

Projeto front-end estático para a página "Meu Perfil" do Belo Eventos.

Conteúdo:

- Arquivos HTML: `index.html`, `login.html`, `meuperfil.html`, `cadastro.html`, `categorias.html`, `comunidade.html`, `asdas.html`.

Como usar

1. Abra os arquivos HTML no seu navegador (duplo clique ou via servidor local).

Inicializar repositório remoto (GitHub)

1. Crie um repositório no GitHub.
2. No seu terminal (PowerShell), dentro do diretório do projeto, execute:

```powershell
# substitua <URL> pelo URL do repositório remoto (HTTPS ou SSH)
git remote add origin <URL>
git branch -M main
git push -u origin main
```
